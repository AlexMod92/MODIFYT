# MODIFYT
MODIFYT is a twin-stick action-sports game where you can choose different types of rackets, goals and balls

## Controls  
Player 1 (Blue)  
Goalkeeper: Left Joystick  
Defense: Right Joystick  
  
Player 2 (Orange)  
Goalkeeper: Right Joystick  
Defense: Left Joystick
